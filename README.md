# 📊 Análise de Feminicídios no Rio de Janeiro (2016–2026)

Este projeto realiza uma análise exploratória de dados sobre feminicídios e tentativas no Rio de Janeiro ao longo de 10 anos.

## 🔍 Objetivos

- Identificar tendências ao longo do tempo
- Detectar regiões mais críticas
- Analisar padrões mensais
- Mapear municípios com maior incidência

## 🛠️ Tecnologias

- Python
- Pandas
- NumPy
- Matplotlib

## 📈 Principais Insights

- Crescimento dos casos ao longo dos anos
- Concentração em regiões específicas (RISP)
- Meses com maior incidência
- Municípios mais afetados
